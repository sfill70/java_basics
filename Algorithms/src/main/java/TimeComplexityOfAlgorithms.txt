Поиск минимального значения в массиве чисел длиной n, который отсортирован по возрастанию.
O(1)
Расчёт среднего значения в массиве чисел длиной n.
O(n)

Получение длины массива размером n.
O(n)

Задан список из n объектов, каждый из которых представляет собой банковский счёт
 - ArrayList<Bill>. И есть класс и метод, с помощью которых можно получить общую
  сумму транзакций между первым и вторым счётом -
  TransactionsCalculator.calculateTotalSum(Bill sourceBill, Bill destinationBill).
  Ваш алгоритм должен, используя этот метод, посчитать общую сумму переводов между
  всеми счетами из списка. Какова будет временная сложность такого алгоритма?

  O(m*n*n/2)
  Где m - клличество транзакций
  n - длинна списка вычисление пойдет в двух циклах, второй вложенный будетт сокращаться, от n до 0
